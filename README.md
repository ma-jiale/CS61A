# cs61a 个人总结

## 1.学习总结

笔者是非计算机专业的本科生，对计算机科学充满兴趣，偶然间看到了[PKUFlyingPig](https://github.com/PKUFlyingPig)大佬的CS自学指南，在指南的前言部分大佬生动地描述了他学习cs61a的体验。我正好想要系统地学习Python语言，于是我便用课余时间来学习cs61a，2024年9月6日开始到2025年12月21日结束，累计472天。除去中间保研和实习占用的时间，粗略算了一下我总共花了约187天学完了所有的内容。

我选择的版本是Fall 2020，此版本由 John DeNero 与 Hany Farid 担任教员。两位老师都和蔼认真，各有风格。

我一般先将当天的Lecture相关的Textbook的相关章节看完，后看Lecture和Q&A, 然后完成对应的Homework、Lab、Discussion，因为Project的内容比较多，我会选择一段完整的时间将它一口气做完。

## 2.参考资料

[Lecture](https://www.bilibili.com/video/BV1s3411G7yM/?p=1&vd_source=bf08880c4c4d8fdcca4d17ed2ee821fe) -- 最完整的带英文字幕的Lecture合集

[half-dreamer](https://github.com/half-dreamer/CS61A-20fa/tree/main/CS61A 2020 FALL all-solution) -- 作者提供了 Fall 2020 习题和solution的备份。

## 3.后记

在上课时，有一件事情使我深受触动，cs61a这门课评分采用积分制，Homework、Lab、Discussion都有对应的分数，并且大部分时候都是有多余分数的，学生只要肯努力肯花时间就能获得正向的反馈。
